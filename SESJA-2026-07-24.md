```
terra-os/
  apps/ui/                    ← Next.js 15, port 3001
    src/
      app/app/page.tsx        ← Hub (useAuthFetch, live stats + recent tenders)
      components/pages/       ← wszystkie strony aplikacji
      lib/api-v2.ts           ← useAuthFetch hook (zwraca parsed JSON, NIE Response!)
      store/useStore.ts       ← Zustand persist (accessToken, user)
  services/api/               ← FastAPI, port 8000
```

### KRYTYCZNY PITFALL — `useAuthFetch`
```ts
// api-v2.ts — hook zwraca res.json() — czyli PARSED OBJECT, nie Response
const data = await authFetch('/api/v2/...');  // ✅
const data = await authFetch('...').then(r => r.json()); // ❌ crash — double parse
```
